## 说明
* company name要求只能填写大小写字母和数字，不可以超过15个字符
* founded year要求只能填写合理四位数字年份，1970年至今
* founder name要求只能填写大小写字母，不可以超过15个字符
* 点击search按钮后先进行validation，若上述条件中有任意一个不符合条件，不显示搜索结果，且在对应不符的条件下红字提示
* 搜索结果以表格形式呈现，单双行用两种颜色区分
* 搜索结果用js代码动态生成，ajax获取companies.json文件
* 搜索结果按company name的字母顺序排序
